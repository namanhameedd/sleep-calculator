# Sleep Calculator

Static site: a sleep-cycle based bedtime/wake-time calculator (client-side only).

## Contents
- `index.html` — main page
- `css/style.css` — styles
- `js/script.js` — logic + cookie consent
- `legal/privacy.html`, `legal/terms.html`
- `images/` — placeholders (favicon, og-image, logo)

## Quick deploy
1. Replace placeholders:
   - `G-XXXXXX` in `index.html` with your Google Analytics ID.
   - `ca-pub-XXXXXXXXXXXX` and `data-ad-slot` with your AdSense IDs.
   - Add real images into `/images/`.
2. Push repo to GitHub.
3. Deploy to GitHub Pages.
